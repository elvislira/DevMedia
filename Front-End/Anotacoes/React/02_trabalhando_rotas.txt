- Usando App Route
    1. Ao usar App Route a pasta pages será substituída por app dentro de src.
    2. A pasta app será responsável por armazenar os arquivos padrão e todas as páginas da aplicação.
    3. A pasta styles deixa de existir e os arquivos de estilos passam a serem armazenados diretamente na pasta app.
    4. O arquivo index.js passa a ser o arquivo page.js na pasta app.
    5. o arquivo Home.module.css passa a se chamar page.module.css.

- Arquivo layout.js
    * Serve para armazenar uma estrutura base a ser usada por todas as páginas da aplicação.

        import Topo from "@/componentes/Topo";
        import Rodape from "@/componentes/Rodape";

        export const metadata = {
            title: "Create Next App",
            description: "Generated by create next app",
            robots: "index, follow",
        };

        export default function RootLayout({ children }) {
            return (
                <html lang="en">
                <body>
                    <Topo />

                    {children}

                    <Rodape />
                </body>
                </html>
            );
        }

- Implementando o App Router manualmente
    1. Abrir o arquivo next.config.mjs alterando o conteúdo da seguinte forma:
        /** @type {import('next').NextConfig} */
        const nextConfig = {
            reactStrictMode: true,
            appRouter: true     --> Indica que vamos usar App Router
        };

        export default nextConfig;
    2. Criar a pasta app na pasta src.
    3. Dentro da pasta app, criar os arquivos:
        * page.js
        * layout.js
        * global.css
        * page.module.css
    4. Criar o conteúdo do arquivo page.js:
        import estilos from "./page.module.css";

        export default function Home() {
            return (
                ...
            );
        }
    5. Criar o conteúdo do arquivo layout.js:
        import "./globals.css";

        export const metadata = {
            title: "Curso Rotas - Exemplo Prático",
            description: "Generated by create next app",
        };

        export default function RootLayout({ children }) {
            return (
                <html lang="en">
                    <body>
                        {children}
                    </body>
                </html>
            );
        }
    6. Podem ser excluídas as pastas pages e styles
