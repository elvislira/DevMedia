- Form
    <form action="url"> </form>
    Obs.: 
        1. Sem o action os dados são enviados para o próprio form.
        2. method="get/post"

- Textarea
    <textarea name="nome_campo"></textarea>

- Button
    <button type="submit">Texto botão</button>

- Label
    <label>Texto</label>
    Atributo:
        for

- Input
    <input type="" name="nome_campo" />
    * Tipos de input (type):
        text
        password
        date
        number
        email
        tel
        checkbox
        radio
    * Atributos do input:
        placeholder
        required
        minlength (text, tel, email, password)
        maxlength (text, tel, email, password)
        min (number, date)
        max (number, date)
        value (checkbox, radio)
        checked (checkbox, radio)
    
    Obs:
        1. Dois ou mais checkbox estarão na mesma categoria se tiverem o mesmo nome seguido de []

- Select
    <select name="nome_campo">
        <option value="">Selecione uma opção</option>
        <option value="valor_campo1">Campo 1</option>
        <option value="valor_campo2">Campo 2</option>
        <option value="valor_campo3">Campo 3</option>
        ...
    </select>

    Atributos:
        selected (option)
        required (select)
        size (select)
