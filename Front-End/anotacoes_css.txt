Propriedades de font:
- font-size (px)
- font-family (Arial, Verdana, Impact, Tahoma, etc...)
- font-weight (normal, bold, 300, 400, ..., 900)
- font-style (italic, normal)
- color (nominal, rgb, hexadecimal)

Propriedades de texto:
- text-align (left, center, right, justify)
- text-decoration (none, underline, line-through, overline)
- line-height (px)

Background:
- background-color (nominal, rgb, hexadecimal)
- background-image url('caminho_imagem')

Estilizando imagem de fundo:
- background-repeat (no-repeat, repeat-x, repeat-y)
- background-position (left, center, right, top, bottom)
- background-size (cover, contain, [width% height%], [widthpx heightpx])
- background-attachment (fixed)

Forma reduzida background:
- background [cor] [url] [repeat] [position] [attachment]

Sites para obter cores:
- Mozilla
- Image color picker

Definindo margens:
- margin (px)
- margin-top
- margin-bottom
- margin-left
- margin-right

Formas compactas de margin:
- margin [top] [right] [bottom] [left]
- margin [top/bottom] [left/right]

Centralizando objeto usando margin:
- margin px auto

Definindo bordas:
- border-width (px)
- border-style (solid, dashed, dotted) obrigatório
- border-color
- border [px] [style] [color] forma padrão
- border [style] [color]
- border [px] [style]
- border [style]
- border-top
- border-bottom
- border-right
- border-left

Definindo espaçamento interno:
- padding (px)
- padding-top
- padding-bottom
- padding-right
- padding-left

Formas compactas de padding:
- padding [top] [right] [bottom] [left]
- padding [top/bottom] [left/right]

Arredondamento de bordas:
- border-radius (px)
- border-radius [top-left] [top-right] [bottom-right] [bottom-left]
- border-radius [top-left/bottom-right] [top-right/bottom-left]

Aplicando dois estilos ao mesmo tempo:
- Separar os elementos por vírgula

Valores relativos à tela
- vw = 1% da largura tela
- vh = 1% da altura da tela

Fazendo cálculos
- calc(expressão a calcular)

Largura mínima e máxima
- min-width (px, %, vw/hw, calc)
- max-width (px, %, vw/hw, calc)

Altura mínima e máxima
- min-height (px, %, vw/hw, calc)
- max-height (px, %, vw/hw, calc)

Exibindo rolagem em objetos
- overflow (scroll, auto)
- overflow-x (scroll, auto)
- overflow-y (scroll, auto)
- overflow [x] [y]

Manipulando o mode exibição de um elemento
- display (inline, block, none)

Manipulando a visibilidade de um elemento
- visibility (visible, hidden)

Manipulando a transparência (opacidade) do elemento
- opacity (0% a 100%)

Alterando a forma como o CSS calcula tamanho de elementos
- box-sizing (border-box, content-box)

Adicionando sombra em volta de elementos
- box-shadow [xpx] [ypx] [desfoque px] [epansão px] [cor] [inset]
