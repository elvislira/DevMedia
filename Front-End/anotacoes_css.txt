Propriedades de font:
- font-size (px)
- font-family (Arial, Verdana, Impact, Tahoma, etc...)
- font-weight (normal, bold, 300, 400, ..., 900)
- font-style (italic, normal)
- color (nominal, rgb, hexadecimal)

Propriedades de texto:
- text-align (left, center, right, justify)
- text-decoration (none, underline, line-through, overline)
- line-height (px)

Background:
- background-color (nominal, rgb, hexadecimal)
- background-image url('caminho_imagem')

Estilizando imagem de fundo:
- background-repeat (no-repeat, repeat-x, repeat-y)
- background-position (left, center, right, top, bottom)
- background-size (cover, contain, [width% height%], [widthpx heightpx])
- background-attachment (fixed)

Forma reduzida background:
- background [cor] [url] [repeat] [position] [attachment]

Sites para obter cores:
- Mozilla
- Image color picker

Definindo margens:
- margin (px)
- margin-top
- margin-bottom
- margin-left
- margin-right

Formas compactas de margin:
- margin [top] [right] [bottom] [left]
- margin [top/bottom] [left/right]

Centralizando objeto usando margin:
- margin px auto

Definindo bordas:
- border-width (px)
- border-style (solid, dashed, dotted) obrigatório
- border-color
- border [px] [style] [color] forma padrão
- border [style] [color]
- border [px] [style]
- border [style]
- border-top
- border-bottom
- border-right
- border-left

Definindo espaçamento interno:
- padding (px)
- padding-top
- padding-bottom
- padding-right
- padding-left

Formas compactas de padding:
- padding [top] [right] [bottom] [left]
- padding [top/bottom] [left/right]

Arredondamento de bordas:
- border-radius (px)
- border-radius [top-left] [top-right] [bottom-right] [bottom-left]
- border-radius [top-left/bottom-right] [top-right/bottom-left]

- Separar os elementos por vírgula
