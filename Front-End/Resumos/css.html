Principais Propriedades

Fontes

font-size

Altera o tamanho da font.

Valores: px.

font-family

Define a família de fonte a ser usada.

Valores: Verdana, Impact, Arial, etc...

font-weight
Define a espessura da fonte. Valores: normal, bold, 100, 200, 300, ..., 900
font-style

Determina o estilo da fonte.

Valores: normal, italic

color

Altera a cor da fonte.

Valores: blue, #0000ff, rgb(0, 0, 255)

Textos

text-align

Define o alinhamento do texto.

Valores: left, center, right, justify

text-decoration

Determina o tipo de linha que terá no texto.

Valores: none, underline, line-through, overline

line-height

Define a altura do texto.

Valores: px

Background

background-color

Altera a cor de fundo do elemento.

Valores: blue, #0000ff, rgb(0, 0, 255)

background-image

Coloca uma imagem ao fundo do elemento.

Valores: url('caminho_imagem')

background-repeat

Determina a repetição ou não da imagem de fundo.

Valores: no-repeat, repeat-x, repeat-y

background-position

Define a posição da imagem de fundo.

Valores: left, center, right, top, bottom

background-size

Determina o tamanho da imagem de fundo.

Valores: cover, contain, [width% height%], [widthpx heightpx]

background-attachment

Determina a rolagem da imagem na tela.

Valores: fixed

background

Usado para definir em uma só linha todas as características definidas anteriormente.

Sintaxe: background [cor] [url] [repeat] [position] [attachment]
